---
title: "Remover Contato"
api: "POST /contact/remove"
description: "Remover um contato da agenda do WhatsApp"
---

Exclui um contato da agenda do WhatsApp conectado, utilizando o sistema de sincronização AppState para atualizar todos os dispositivos vinculados.

## Funcionalidades de Remoção

<AccordionGroup>
  <Accordion title="Sincronização AppState" icon="cloud-arrow-up">
    A remoção utiliza o sistema nativo de sincronização do WhatsApp.

    **Comportamento:**
    - Remove da agenda local do WhatsApp
    - Sincroniza com WhatsApp Web automaticamente
    - Atualiza aplicativo móvel em tempo real
    - Mantém histórico de conversas intacto

    **Importante:** A remoção afeta apenas a agenda, conversas anteriores são preservadas.

  </Accordion>

  <Accordion title="Formatos de Número Aceitos" icon="phone">
    O endpoint aceita diversos formatos de identificação.

    **Formatos suportados:**
    - Número simples: `5511999999999`
    - Com símbolo +: `+5511999999999`
    - Com formatação: `+55 (11) 99999-9999`
    - Formato JID: `5511999999999@s.whatsapp.net`

    **Obrigatório:** Código do país (Brasil = 55)

  </Accordion>

  <Accordion title="Restrições de Tipo" icon="ban">
    Alguns tipos de identificadores não podem ser removidos.

    **Não aceita:**
    - Contatos comerciais (`@lid`)
    - Grupos (`@g.us`)
    - Comunidades e canais
    - Números inválidos ou inexistentes

    Apenas contatos pessoais salvos na agenda são elegíveis para remoção.

  </Accordion>

  <Accordion title="Impacto da Remoção" icon="info-circle">
    Entenda o que acontece após remover um contato.

    **Efeitos:**
    - Contato não aparece mais na agenda
    - Conversas anteriores permanecem disponíveis
    - Número aparece sem nome nas mensagens
    - É possível re-adicionar o contato posteriormente

    **Nota:** A remoção não bloqueia nem deleta conversas existentes.

  </Accordion>
</AccordionGroup>

### Authorization

<ParamField path="api_key" type="string" required>
  Chave de autenticação da sua conexão, utilizada no header Authorization como
  **Bearer** Token. <br />
  Você encontra sua API Key no botão **API** da sua conexão no painel ZapClass. <br />
  A chave sempre começa com o prefixo `zc_`. <br />
  Exemplo: `zc_6FrX2qjM4bHT9pR6cC0sD8wLkG3nE2tZfV5uN1aP`
</ParamField>

### Body Parameters

<ParamField path="phone" type="string" required>
  Número de telefone no formato internacional. <br />
  Deve incluir código do país (Brasil = 55). <br />
  Aceita diversos formatos com/sem símbolos, parênteses, hífen ou espaços.{" "}
  <br />
  Também aceita formato JID (@s.whatsapp.net). <br />
  Exemplo: `5511999999999` ou `+55 (11) 99999-9999`
</ParamField>

<Warning>
  **Atenção:** A remoção é permanente. Para adicionar novamente, será necessário
  usar o endpoint `/contact/add`.
</Warning>

<RequestExample>

```bash cURL
curl -X POST https://zapclass.com.br/api/contact/remove \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer zc_6FrX2qjM4bHT9pR6cC0sD8wLkG3nE2tZfV5uN1aP" \
  -d '{
    "phone": "5511999999999"
  }'
```

```javascript JavaScript
const response = await fetch("https://zapclass.com.br/api/contact/remove", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    Authorization: "Bearer zc_6FrX2qjM4bHT9pR6cC0sD8wLkG3nE2tZfV5uN1aP",
  },
  body: JSON.stringify({
    phone: "5511999999999",
  }),
});

const data = await response.json();
if (data.success) {
  console.log("Contato removido com sucesso");
} else {
  console.error("Erro ao remover:", data.message);
}
```

```php PHP
<?php
$ch = curl_init('https://zapclass.com.br/api/contact/remove');

$payload = [
    "phone" => "5511999999999"
];

curl_setopt($ch, CURLOPT_POST, 1);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($payload));
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Content-Type: application/json',
    'Authorization: Bearer zc_6FrX2qjM4bHT9pR6cC0sD8wLkG3nE2tZfV5uN1aP'
]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$data = json_decode($response, true);

if ($data['success']) {
    echo "Contato {$data['removedContact']['phone']} removido com sucesso";
} else {
    echo "Erro: " . $data['message'];
}
curl_close($ch);
?>
```

</RequestExample>
