---
title: "Enviar Menu Interativo"
api: "POST /messages/menu"
description: "Enviar menu interativo (botões, carrossel, lista ou enquete)"
---

Este endpoint disponibiliza uma interface unificada para envio de quatro tipos principais de mensagens interativas: **Botões**, **Carrossel de Botões**, **Listas** e **Enquetes**.

## Tipos de Mensagens Interativas

<AccordionGroup>
  <Accordion title="Botões (type: button)" icon="rectangle-wide">
    Cria botões interativos para ações rápidas e diretas.

    **Formatos suportados:**
    - Botão de Resposta: `"texto|id"` ou `"texto"`
    - Botão de Cópia: `"texto|copy:código"`
    - Botão de Chamada: `"texto|call:+5511999999999"`
    - Botão de URL: `"texto|https://exemplo.com"`

    **Campo especial:** `imageButton` para adicionar imagem aos botões
  </Accordion>

  <Accordion title="Listas (type: list)" icon="list-check">
    Cria menus organizados em seções com itens selecionáveis.

    **Formato das opções:**
    - `"[Título da Seção]"` - Inicia uma nova seção
    - `"texto|id|descrição"` - Item da lista

    **Campos especiais:** `listButton` (texto do botão), `footerText` (rodapé)
  </Accordion>

  <Accordion title="Enquetes (type: poll)" icon="chart-bar">
    Cria enquetes interativas para votação.

    **Campo especial:** `selectableCount` - número de opções selecionáveis (padrão: 1)

    As `choices` são um array simples com as opções de voto
  </Accordion>

  <Accordion title="Carrossel (type: carousel)" icon="images">
    Cria um carrossel de cartões com imagens e botões.

    **Formato das choices (em ordem):**
    - `"[Texto do cartão]"` - Texto entre colchetes
    - `"{URL ou base64}"` - Imagem entre chaves
    - Botões do cartão (copy, url ou call)
  </Accordion>
</AccordionGroup>

### Authorizations

<ParamField path="token" type="string" required>
  Token de autenticação de conexão com a API. <br />
  Exemplo: `zt_BppKZWtzvj1X2SbJAuOSkA0Z63uDBQ0wkb28NJaloKeOUnHk`
</ParamField>

### Body Parameters

<ParamField path="number" type="string" required>
  Número do destinatário no formato internacional. <br />
  Deve incluir código do país DDI + DDD + número. <br />
  Exemplo: `5511999999999`
</ParamField>

<ParamField path="type" type="string" required>
  Tipo do menu interativo. <br />
  Opções: `button`, `list`, `poll`, `carousel` <br />
  Exemplo: `list`
</ParamField>

<ParamField path="text" type="string" required>
  Texto principal da mensagem (aceita placeholders). <br />
  Exemplo: `Escolha uma opção:`
</ParamField>

<ParamField path="choices" type="array" required>
  Lista de opções do menu. <br />
  Use `[Título]` para seções em listas. <br />
  Formato varia conforme o tipo escolhido. <br />
  Exemplo: `["[Eletrônicos]","Smartphones|phones|Últimos lançamentos"]`
</ParamField>

<ParamField path="footerText" type="string">
  Texto do rodapé (opcional para botões e listas). <br />
  Exemplo: `Menu de serviços`
</ParamField>

<ParamField path="listButton" type="string">
  Texto do botão principal (para listas). <br />
  Exemplo: `Ver opções`
</ParamField>

<ParamField path="selectableCount" type="integer">
  Número máximo de opções selecionáveis (apenas para enquetes). <br />
  Padrão: `1` <br />
  Exemplo: `1`
</ParamField>

<ParamField path="imageButton" type="string">
  URL ou Base64 da imagem para botões (recomendado para type: button). <br />
  Exemplo: `https://exemplo.com/imagem-botao.jpg`
</ParamField>

<ParamField path="delay" type="integer">
  Atraso em milissegundos antes do envio. <br />
  Durante o atraso aparecerá "Digitando...". <br />
  Exemplo: `1000`
</ParamField>

<ParamField path="replyid" type="string">
  ID da mensagem para responder. <br />
  Exemplo: `3EB0538DA65A59F6D8A251`
</ParamField>

<ParamField path="mentions" type="string">
  Números para mencionar (separados por vírgula). <br />
  Exemplo: `5511999999999,5511888888888`
</ParamField>

<ParamField path="readchat" type="boolean">
  Marca conversa como lida após envio. <br />
  Exemplo: `true`
</ParamField>

<ParamField path="readmessages" type="boolean">
  Marca últimas mensagens recebidas como lidas. <br />
  Exemplo: `true`
</ParamField>

<ParamField path="track_source" type="string">
  Origem do rastreamento da mensagem. <br />
  Exemplo: `chatwoot`
</ParamField>

<ParamField path="track_id" type="string">
  ID para rastreamento da mensagem (aceita valores duplicados). <br />
  Exemplo: `msg_123456789`
</ParamField>

<Warning>
  **Importante:** Ao combinar botões de resposta com outros tipos (call, url, copy) na mesma mensagem, será exibido aviso no WhatsApp Web para visualizar no celular.
</Warning>
