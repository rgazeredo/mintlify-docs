---
title: "Enviar Texto"
api: "POST /messages/text"
description: "Enviar mensagem de texto simples"
---

### Authorization

<ParamField path="api_key" type="string" required>
  Chave de autenticação da sua conexão, utilizada no header Authorization como
  **Bearer** Token. <br />
  Você encontra sua API Key no botão **API** da sua conexão no painel ZapClass. <br />
  A chave sempre começa com o prefixo `zc_`. <br />
  Exemplo: `zc_6FrX2qjM4bHT9pR6cC0sD8wLkG3nE2tZfV5uN1aP`
</ParamField>

### Body Parameters

<ParamField path="number" type="string" required>
  Número do destinatário no formato internacional. <br />
  Deve incluir código do país DDI + DDD + número. <br />
  Exemplo: `5511999999999`
</ParamField>

<ParamField path="message" type="string" required>
  Conteúdo da mensagem de texto a ser enviada. <br />
  Suporta texto simples, emojis e caracteres especiais. <br />
  Exemplo: `Olá! Bem-vindo à nossa plataforma ZapClass`
</ParamField>

<ParamField path="delay" type="integer">
  Atraso em milissegundos antes do envio (agendamento). <br />
  Útil para envios programados ou intervalos entre mensagens. <br />
  Aparecerá "Digitando..." antes do envio. <br />
  Exemplo: `3000`
</ParamField>

<ParamField path="forward" type="boolean">
  Marca a mensagem como encaminhada para o destinatário. <br />
  Exemplo: `true`
</ParamField>

<ParamField path="link_preview" type="boolean">
  Controla se links na mensagem devem exibir prévia automática. <br />
  true: exibe prévia, false: apenas o link. <br />
  Exemplo: `true`
</ParamField>

<ParamField path="link_preview_title" type="string">
  Título personalizado para prévia de link. <br />
  Se fornecido, será exibido como o título da prévia do link. <br />
  Limite: `60 caracteres` <br />
  Exemplo: `Novidade! Loja online com frete grátis em todo o Brasil`
</ParamField>

<ParamField path="link_preview_description" type="string">
  Descrição personalizada para prévia de link. <br />
  Se fornecido, será exibido como a descrição da prévia do link. <br />
  Limite: `80 caracteres` <br />
  Exemplo: `Aproveite promoções exclusivas e receba seus produtos sem custo de entrega.`
</ParamField>

<ParamField path="link_preview_image" type="string">
  URL ou Base64da imagem personalizada para prévia de link. <br />
  Se fornecido, será exibido como a imagem da prévia do link. <br />
  Exemplo: `https://placehold.co/600x400.png`
</ParamField>

<ParamField path="link_preview_large" type="boolean">
  Controla se a prévia de link deve ser exibida em tamanho grande. <br />
  true: exibe em tamanho grande, false: exibe em tamanho padrão <br />
  Exemplo: `true`
</ParamField>

<ParamField path="message_reply_id" type="string">
  Identificador da mensagem para responder. <br />
  Exemplo: `5511999999999:1AB2C3DEFG4H56I78JKL9M`
</ParamField>

<ParamField path="message_source" type="string">
  Origem do envio da mensagem. <br />
  Normalmente usado para rastreamento da mensagem. <br />
  Exemplo: `typebot`
</ParamField>

<ParamField path="message_id" type="string">
  Identificador do envio da mensagem. <br />
  Combinado com o `message_source` pode ser usado para rastreamento. <br />
  Exemplo: `5511999999999:1AB2C3DEFG4H56I78JKL9M`
</ParamField>

<ParamField path="mentions" type="string">
  Relação de números de telefone a serem mencionados na mensagem. <br />
  O números devem ser separads por vírgula. <br />
  Exemplo: `5511999999999,5511888888888`
</ParamField>

<ParamField path="read" type="boolean">
  Marca a conversa como lida após o envio. <br />
  Exemplo: `true`
</ParamField>

<ParamField path="read_messages" type="boolean">
  Marca as últimas mensagens como lidas após o envio. <br />
  Exemplo: `true`
</ParamField>
