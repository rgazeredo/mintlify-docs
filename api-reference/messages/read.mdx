---
title: "Marcar como Lidas"
api: "POST /message/read"
description: "Marcar uma ou mais mensagens como lidas"
---

Marca mensagens recebidas como lidas, atualizando o status de visualização no WhatsApp e sincronizando entre dispositivos conectados.

## Funcionalidades

<AccordionGroup>
  <Accordion title="Marcação em Lote" icon="check-double">
    Processe múltiplas mensagens simultaneamente enviando um array de IDs.

    **Benefícios:**
    - Otimiza requisições à API
    - Atualiza status de várias conversas de uma vez
    - Reduz tempo de processamento

  </Accordion>

  <Accordion title="Sincronização Automática" icon="arrows-rotate">
    O status de leitura é sincronizado automaticamente com todos os dispositivos vinculados.

    **Comportamento:**
    - WhatsApp Web atualiza em tempo real
    - Aplicativo mobile recebe notificação
    - Status visível para o remetente

  </Accordion>

  <Accordion title="Confirmação de Leitura" icon="eye">
    Cada mensagem marcada retorna informação de timestamp confirmando a ação.

    **Resposta inclui:**
    - ID da mensagem processada
    - Timestamp de quando foi marcada como lida
    - Status de sucesso individual

  </Accordion>
</AccordionGroup>

### Authorization

<ParamField path="api_key" type="string" required>
  Chave de autenticação da sua conexão, utilizada no header Authorization como
  **Bearer** Token. <br />
  Você encontra sua API Key no botão **API** da sua conexão no painel ZapClass. <br />
  A chave sempre começa com o prefixo `zc_`. <br />
  Exemplo: `zc_6FrX2qjM4bHT9pR6cC0sD8wLkG3nE2tZfV5uN1aP`
</ParamField>

### Body Parameters

<ParamField path="ids" type="array" required>
  Lista de IDs das mensagens a serem marcadas como lidas. <br />
  Aceita um ou múltiplos IDs de mensagens. <br />
  Exemplo:
  `["5511999999999:1AB2C3DEFG4H56I78JKL9M","5511999999999:2AB2C3DEFG4H56I78JKL9M"]`
</ParamField>

<Tip>
  **Dica:** Você pode marcar até 50 mensagens por requisição para otimizar o
  desempenho.
</Tip>

<RequestExample>

```bash cURL

  -H "Content-Type: application/json" \
  -H "Authorization: Bearer zc_6FrX2qjM4bHT9pR6cC0sD8wLkG3nE2tZfV5uN1aP" \
  -d '{
    "ids": [
      "5511999999999:1AB2C3DEFG4H56I78JKL9M",
      "5511999999999:2AB2C3DEFG4H56I78JKL9M",
    ]
  }'
```
