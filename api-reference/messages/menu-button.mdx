---
title: "Enviar Botões Interativos"
api: "POST /messages/menu-button"
description: "Enviar mensagem com botões de ação rápida (resposta, cópia, chamada ou URL)"
---

Botões interativos permitem que o usuário execute ações rápidas diretamente na conversa. Você pode criar até 3 botões por mensagem com diferentes tipos de ação.

## Tipos de Botões

<AccordionGroup>
  <Accordion title="Botão de Resposta" icon="reply">
    Envia uma resposta de texto quando clicado.

    **Formato:** `"texto"` ou `"texto|id"`

    Exemplo: `"Sim, aceito|aceito"` ou `"Não, obrigado"`

  </Accordion>

  <Accordion title="Botão de Cópia" icon="copy">
    Copia um código ou texto para a área de transferência.

    **Formato:** `"texto|copy:código"`

    Exemplo: `"Copiar Cupom|copy:DESCONTO20"`

  </Accordion>

  <Accordion title="Botão de Chamada" icon="phone">
    Inicia uma chamada telefônica.

    **Formato:** `"texto|call:+número"`

    Exemplo: `"Ligar Agora|call:+5511999999999"`

  </Accordion>

  <Accordion title="Botão de URL" icon="link">
    Abre um link no navegador.

    **Formato:** `"texto|https://url"`

    Exemplo: `"Ver Site|https://zapclass.com.br"`

  </Accordion>
</AccordionGroup>

<Warning>
  **Importante:** Ao combinar botões de resposta com outros tipos (call, url,
  copy) na mesma mensagem, será exibido aviso no WhatsApp Web para visualizar no
  celular.
</Warning>

### Authorization

<ParamField path="api_key" type="string" required>
  Chave de autenticação da sua conexão, utilizada no header Authorization como
  **Bearer** Token. <br />
  Você encontra sua API Key no botão **API** da sua conexão no painel ZapClass. <br />
  A chave sempre começa com o prefixo `zc_`. <br />
  Exemplo: `zc_6FrX2qjM4bHT9pR6cC0sD8wLkG3nE2tZfV5uN1aP`
</ParamField>

### Body Parameters

<ParamField path="number" type="string" required>
  Número do destinatário no formato internacional. <br />
  Deve incluir código do país DDI + DDD + número. <br />
  Exemplo: `5511999999999`
</ParamField>

<ParamField path="title" type="string" required>
  Título principal da lista exibido acima do botão. <br />
  Exemplo: `Confira nosso catálogo completo`
</ParamField>

<ParamField path="choices" type="array" required>
  Lista de botões interativos. <br />
  Exemplo: `["Falar com Atendente|atendente", "Ver Catálogo|https://zapclass.com.br",
  "Copiar Cupom|copy:DESC10"]`
</ParamField>

<ParamField path="button_image" type="string">
  URL ou Base64 da imagem exibida acima dos botões. <br />
  Recomendado para deixar a mensagem mais visual. <br />
  Exemplo: `https://placehold.co/600x400.png`
</ParamField>

<ParamField path="footer_text" type="string">
  Texto do rodapé exibido abaixo dos botões. <br />
  Exemplo: `Atendimento disponível 24/7`
</ParamField>

<ParamField path="delay" type="integer">
  Atraso em milissegundos antes do envio. <br />
  Durante o atraso aparecerá "Digitando...". <br />
  Exemplo: `1000`
</ParamField>

<ParamField path="message_reply_id" type="string">
  Identificador da mensagem para responder. <br />
  Exemplo: `5511999999999:1AB2C3DEFG4H56I78JKL9M`
</ParamField>

<ParamField path="message_source" type="string">
  Origem do envio da mensagem. <br />
  Normalmente usado para rastreamento da mensagem. <br />
  Exemplo: `typebot`
</ParamField>

<ParamField path="message_id" type="string">
  Identificador do envio da mensagem. <br />
  Combinado com o `message_source` pode ser usado para rastreamento. <br />
  Exemplo: `5511999999999:1AB2C3DEFG4H56I78JKL9M`
</ParamField>

<ParamField path="mentions" type="string">
  Relação de números de telefone a serem mencionados na mensagem. <br />
  O números devem ser separads por vírgula. <br />
  Exemplo: `5511999999999,5511888888888`
</ParamField>

<ParamField path="read" type="boolean">
  Marca a conversa como lida após o envio. <br />
  Exemplo: `true`
</ParamField>

<ParamField path="read_messages" type="boolean">
  Marca as últimas mensagens como lidas após o envio. <br />
  Exemplo: `true`
</ParamField>

<RequestExample>

```bash cURL
curl -X POST https://zapclass.com.br/api/messages/menu-button \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer zc_6FrX2qjM4bHT9pR6cC0sD8wLkG3nE2tZfV5uN1aP" \
  -d '{
    "number": "5511999999999",
    "title": "Promoção exclusiva! Use o cupom DESCONTO20",
    "choices": [
      "Copiar Cupom|copy:DESCONTO20",
      "Ver Produtos|https://zapclass.com.br",
      "Ligar para Loja|call:+5511999999999"
    ],
    "footer_text": "Válido até 31/12/2024",
    "button_image": "https://placehold.co/600x400.png",
  }'
```

```php PHP
<?php
$ch = curl_init('https://zapclass.com.br/api/messages/menu-button');

$payload = [
    "number" => "5511999999999",
    "title" => "Promoção exclusiva! Use o cupom DESCONTO20",
    "choices" => [
        "Copiar Cupom|copy:DESCONTO20",
        "Ver Produtos|https://zapclass.com.br",
        "Ligar para Loja|call:+5511999999999"
    ],
    "footer_text" => "Válido até 31/12/2024",
    "button_image" => "https://placehold.co/600x400.png",
];

curl_setopt($ch, CURLOPT_POST, 1);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($payload));
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Content-Type: application/json',
    'Authorization: Bearer zc_6FrX2qjM4bHT9pR6cC0sD8wLkG3nE2tZfV5uN1aP'
]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
curl_close($ch);
?>
```
