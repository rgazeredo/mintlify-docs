---
title: "Configurar Campos Personalizados"
api: "POST /instance/updateFieldsMap"
description: "Definir mapeamento de campos personalizados para leads"
---

Configura o mapeamento de campos personalizados (custom fields) da instância. Permite definir até 20 campos adicionais para armazenar informações específicas sobre leads no CRM integrado.

## Sobre Campos Personalizados

<AccordionGroup>
  <Accordion title="O que são Campos Personalizados" icon="table-columns">
    Sistema flexível para armazenar dados adicionais de leads.

    **Características:**
    - Até 20 campos personalizáveis (lead_field01 a lead_field20)
    - Cada campo aceita até 255 caracteres
    - Armazena qualquer tipo de dado em formato string
    - Totalmente configurável por instância
    - Útil para integração com CRM e sistemas externos

    **Funcionamento:**
    - Configure o mapeamento uma vez
    - Use os campos ao criar/atualizar leads
    - Acesse os dados em qualquer consulta de lead
    - Campos não configurados permanecem vazios
    - Atualização parcial preserva campos não enviados
  </Accordion>

  <Accordion title="Casos de Uso Comuns" icon="lightbulb">
    Aplicações práticas para campos personalizados.

    **Informações de Contato Estendidas:**
    - Nome completo, apelido, cargo
    - Email secundário, telefone comercial
    - Endereço completo, CEP, bairro
    - Redes sociais, website

    **Dados de Qualificação:**
    - Score de lead, temperatura (quente/frio)
    - Fonte de captação, campanha de origem
    - Produto de interesse, ticket médio
    - Fase do funil, probabilidade de conversão

    **Gestão Comercial:**
    - Vendedor responsável, equipe
    - Última interação, próximo follow-up
    - Histórico de compras, valor LTV
    - Observações, anotações importantes

    **Segmentação:**
    - Tags, categorias, grupos
    - Setor, porte da empresa
    - Região, território de vendas
    - Perfil comportamental
  </Accordion>

  <Accordion title="Configuração e Mapeamento" icon="gear">
    Como estruturar seus campos personalizados.

    **Planejamento:**
    - Liste todas as informações necessárias
    - Agrupe dados relacionados
    - Defina nomes descritivos e padronizados
    - Reserve campos para expansão futura

    **Boas Práticas:**
    - Use nomes em português e descritivos
    - Mantenha consistência na nomenclatura
    - Documente o propósito de cada campo
    - Evite abreviações confusas
    - Planeje antes de implementar

    **Exemplos de Mapeamento:**
    - lead_field01: "nome_completo"
    - lead_field02: "email"
    - lead_field03: "telefone"
    - lead_field04: "cidade"
    - lead_field05: "estado"
    - lead_field06: "cargo"
  </Accordion>

  <Accordion title="Integração com CRM" icon="diagram-project">
    Como usar campos personalizados em sistemas integrados.

    **Sincronização:**
    - Mapear campos do CRM para lead_fieldXX
    - Importar dados de sistemas externos
    - Exportar informações completas de leads
    - Manter sincronia bidirecional

    **Automação:**
    - Disparar ações baseadas em valores de campos
    - Segmentar audiência por critérios personalizados
    - Personalizar mensagens com dados dos campos
    - Criar relatórios e dashboards customizados

    **Exemplos de Integração:**
    - Pipedrive, RD Station, HubSpot
    - Planilhas Google Sheets, Excel
    - Sistemas ERP internos
    - Plataformas de e-commerce
  </Accordion>
</AccordionGroup>

```bash cURL
curl -X POST https://zapclass.com.br/api/instance/updateFieldsMap \
  -H "Content-Type: application/json" \
  -H "Token: zt_BppKZWtzvj1X2SbJAuOSkA0Z63uDBQ0wkb28NJaloKeOUnHk" \
  -d '{
    "lead_field01": "nome_completo",
    "lead_field02": "email",
    "lead_field03": "telefone",
    "lead_field04": "cidade",
    "lead_field05": "estado"
```

```javascript JavaScript
const response = await fetch("https://zapclass.com.br/api/instance/updateFieldsMap", {
  method: "POST",
  headers: {
```

```php PHP
<?php
$ch = curl_init('https://zapclass.com.br/api/instance/updateFieldsMap');

$payload = [
    "lead_field01" => "nome_completo",
    "lead_field02" => "email",
    "lead_field03" => "telefone",
    "lead_field04" => "cidade",
    "lead_field05" => "estado"
];

curl_setopt($ch, CURLOPT_POST, 1);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($payload));
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Content-Type: application/json',
    'Token: zt_BppKZWtzvj1X2SbJAuOSkA0Z63uDBQ0wkb28NJaloKeOUnHk'
]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$data = json_decode($response, true);

if ($data['success']) {
    echo "Campos configurados:\n";
    print_r($data['instance']['fieldsMap']);
} else {
    echo "Erro: " . $data['message'];
}

curl_close($ch);
?>
```
