---
title: "Listar Respostas Rápidas"
api: "GET /quickreply/showall"
description: "Obter todas as respostas rápidas cadastradas"
---

Retorna a lista completa de templates de respostas rápidas cadastrados para a instância autenticada, incluindo templates criados pela API e pelo WhatsApp.

## Sobre Listagem de Respostas Rápidas

<AccordionGroup>
  <Accordion title="Visão Geral da Listagem" icon="list">
    Como funciona a listagem de templates.

    **Retorno:**
    - Array com todos os templates cadastrados
    - Templates criados pela API e pelo WhatsApp
    - Informações completas de cada template
    - Ordenação por data de criação (mais recentes primeiro)
    - Dados em tempo real da instância

    **Informações incluídas:**
    - ID único do template
    - Atalho (shortCut)
    - Tipo de mensagem (text, image, video, etc)
    - Conteúdo (texto, arquivo, nome do documento)
    - Origem (API ou WhatsApp)
    - Data de criação e última atualização
  </Accordion>

  <Accordion title="Templates da API vs WhatsApp" icon="code-compare">
    Diferenças entre templates de diferentes origens.

    **Templates criados pela API:**
    - Campo `onWhatsApp: false`
    - Podem ser editados e excluídos
    - Criados via endpoint /quickreply/create
    - Totalmente gerenciáveis via API
    - Personalizáveis conforme necessidade

    **Templates do WhatsApp:**
    - Campo `onWhatsApp: true`
    - Apenas leitura via API
    - Não podem ser editados ou excluídos
    - Criados diretamente no app WhatsApp
    - Sincronizados automaticamente

    **Identificação:**
    - Verifique o campo `onWhatsApp` para saber a origem
    - Templates do WhatsApp têm ícone especial
  </Accordion>

  <Accordion title="Filtragem e Organização" icon="filter">
    Como organizar e filtrar templates.

    **Filtragem do lado do cliente:**
    - Por tipo de mensagem (text, image, video, etc)
    - Por origem (API ou WhatsApp)
    - Por data de criação
    - Por atalho ou conteúdo (busca textual)
    - Templates editáveis vs somente leitura

    **Organização sugerida:**
    - Agrupar por categoria de uso
    - Separar templates ativos de inativos
    - Ordenar por frequência de uso
    - Criar índice de atalhos
    - Manter biblioteca documentada
  </Accordion>

  <Accordion title="Casos de Uso" icon="lightbulb">
    Situações práticas para listar templates.

    **Aplicações:**
    - Exibir biblioteca de templates em interface
    - Sincronizar templates entre sistemas
    - Backup e exportação de templates
    - Auditoria de templates cadastrados
    - Validação antes de criar novo template
    - Buscar ID de template específico
    - Análise de uso e performance
    - Documentação automática de templates

    **Integração:**
    - Carregar templates em seletores/dropdowns
    - Autocompletar atalhos em chat
    - Dashboard de gerenciamento
    - Relatórios de utilização
  </Accordion>
</AccordionGroup>

### Authorizations

<ParamField path="token" type="string" required>
  Token de autenticação de conexão com a API. <br />
  Exemplo: `zt_BppKZWtzvj1X2SbJAuOSkA0Z63uDBQ0wkb28NJaloKeOUnHk`
</ParamField>

<Info>
  Este endpoint não requer parâmetros adicionais. Basta fornecer o token de autenticação no header.
</Info>

<RequestExample>
```bash cURL
curl -X GET https://zapclass.com.br/api/quickreply/showall \
  -H "Token: zt_BppKZWtzvj1X2SbJAuOSkA0Z63uDBQ0wkb28NJaloKeOUnHk"
```

```javascript JavaScript
const response = await fetch("https://zapclass.com.br/api/quickreply/showall", {
  method: "GET",
  headers: {
    Token: "zt_BppKZWtzvj1X2SbJAuOSkA0Z63uDBQ0wkb28NJaloKeOUnHk",
  },
});

const data = await response.json();
if (data.success) {
  console.log(`Total de templates: ${data.quickReplies.length}`);
  data.quickReplies.forEach(qr => {
    console.log(`- ${qr.shortCut} (${qr.type})`);
  });
}
```

```php PHP
<?php
$ch = curl_init('https://zapclass.com.br/api/quickreply/showall');

curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Token: zt_BppKZWtzvj1X2SbJAuOSkA0Z63uDBQ0wkb28NJaloKeOUnHk'
]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$data = json_decode($response, true);

if ($data['success']) {
    echo "Total de templates: " . count($data['quickReplies']) . "\n";

    foreach ($data['quickReplies'] as $qr) {
        echo "- {$qr['shortCut']} ({$qr['type']})\n";
    }
} else {
    echo "Erro: " . $data['message'];
}

curl_close($ch);
?>
```
</RequestExample>
