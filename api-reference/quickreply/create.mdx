---
title: "Criar Resposta Rápida"
api: "POST /quickreply/create"
description: "Criar novo template de resposta rápida"
---

Cria um novo template de resposta rápida para agilizar o atendimento. Suporta mensagens de texto e diversos tipos de mídia.

## Sobre Respostas Rápidas

<AccordionGroup>
  <Accordion title="O que são Respostas Rápidas" icon="bolt">
    Templates prontos para acelerar o atendimento.

    **Características:**
    - Mensagens pré-configuradas reutilizáveis
    - Acesso via atalho personalizado
    - Suporta texto e mídia
    - Economiza tempo no atendimento
    - Padroniza comunicação da equipe

    **Benefícios:**
    - Respostas instantâneas a perguntas frequentes
    - Consistência nas mensagens
    - Redução de erros de digitação
    - Maior produtividade da equipe
  </Accordion>

  <Accordion title="Tipos de Mensagem Suportados" icon="message">
    Formatos disponíveis para templates.

    **Tipos aceitos:**
    - `text`: Mensagem de texto simples
    - `image`: Imagem com legenda opcional
    - `video`: Vídeo com legenda opcional
    - `audio`: Áudio ou nota de voz
    - `document`: Arquivo PDF, DOC, etc

    **Para cada tipo:**
    - Texto: campo `text` obrigatório
    - Mídia: campo `file` obrigatório (URL ou base64)
    - Documento: campo `docName` opcional para nome do arquivo
  </Accordion>

  <Accordion title="Atalhos (Shortcuts)" icon="keyboard">
    Como funcionam os atalhos de acesso rápido.

    **Definição:**
    - Código único para identificar o template
    - Usado para buscar e enviar rapidamente
    - Deve ser descritivo e memorável
    - Apenas letras, números e underscore

    **Boas práticas:**
    - Use nomes curtos e descritivos
    - Exemplos: `saudacao`, `horario`, `catalogo`
    - Evite caracteres especiais
    - Mantenha padrão de nomenclatura
  </Accordion>

  <Accordion title="Casos de Uso" icon="lightbulb">
    Situações práticas para usar respostas rápidas.

    **Aplicações:**
    - Saudações de boas-vindas
    - Informações de horário de funcionamento
    - Links para catálogos e menus
    - Instruções de pagamento
    - Políticas de troca e devolução
    - Perguntas frequentes (FAQ)
    - Mensagens de ausência
    - Agradecimentos e despedidas
  </Accordion>
</AccordionGroup>

### Authorizations

<ParamField path="token" type="string" required>
  Token de autenticação de conexão com a API. <br />
  Exemplo: `zt_BppKZWtzvj1X2SbJAuOSkA0Z63uDBQ0wkb28NJaloKeOUnHk`
</ParamField>

### Body Parameters

<ParamField path="shortCut" type="string" required>
  Atalho para acesso rápido ao template. <br />
  Apenas letras, números e underscore. <br />
  Exemplo: `"saudacao1"`
</ParamField>

<ParamField path="type" type="string" required>
  Tipo da mensagem. <br />
  **Valores:** `text`, `image`, `video`, `audio`, `document` <br />
  Exemplo: `"text"`
</ParamField>

<ParamField path="text" type="string">
  Conteúdo da mensagem de texto. <br />
  Obrigatório para tipo `text`. <br />
  Opcional como legenda para tipos de mídia. <br />
  Exemplo: `"Olá! Como posso ajudar hoje?"`
</ParamField>

<ParamField path="file" type="string">
  URL ou Base64 do arquivo. <br />
  Obrigatório para tipos de mídia. <br />
  Exemplo: `"https://exemplo.com/arquivo.pdf"`
</ParamField>

<ParamField path="docName" type="string">
  Nome do arquivo para tipo documento. <br />
  Opcional, usado apenas com `type: document`. <br />
  Exemplo: `"apresentacao.pdf"`
</ParamField>

<Info>
  **Dica:** Use atalhos descritivos como `saudacao`, `horario`, `catalogo` para facilitar a busca e uso dos templates.
</Info>

<RequestExample>
```bash cURL
curl -X POST https://zapclass.com.br/api/quickreply/create \
  -H "Content-Type: application/json" \
  -H "Token: zt_BppKZWtzvj1X2SbJAuOSkA0Z63uDBQ0wkb28NJaloKeOUnHk" \
  -d '{
    "shortCut": "saudacao",
    "type": "text",
    "text": "Olá! Como posso ajudar você hoje?"
  }'
```

```javascript JavaScript
const response = await fetch("https://zapclass.com.br/api/quickreply/create", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    Token: "zt_BppKZWtzvj1X2SbJAuOSkA0Z63uDBQ0wkb28NJaloKeOUnHk",
  },
  body: JSON.stringify({
    shortCut: "saudacao",
    type: "text",
    text: "Olá! Como posso ajudar você hoje?"
  }),
});

const data = await response.json();
if (data.success) {
  console.log("Resposta rápida criada:", data.quickReply.id);
}
```

```php PHP
<?php
$ch = curl_init('https://zapclass.com.br/api/quickreply/create');

$payload = [
    "shortCut" => "saudacao",
    "type" => "text",
    "text" => "Olá! Como posso ajudar você hoje?"
];

curl_setopt($ch, CURLOPT_POST, 1);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($payload));
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Content-Type: application/json',
    'Token: zt_BppKZWtzvj1X2SbJAuOSkA0Z63uDBQ0wkb28NJaloKeOUnHk'
]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$data = json_decode($response, true);

if ($data['success']) {
    echo "Resposta rápida criada: {$data['quickReply']['id']}\n";
} else {
    echo "Erro: " . $data['message'];
}

curl_close($ch);
?>
```
</RequestExample>
