---
title: "Criar Grupo"
api: "POST /group/create"
description: "Criar um novo grupo no WhatsApp com participantes iniciais"
---

Cria um novo grupo no WhatsApp adicionando participantes iniciais e configurando as informações básicas do grupo.

## Detalhes de Criação

<AccordionGroup>
  <Accordion title="Processo de Criação" icon="users">
    O grupo é criado com você como administrador e os participantes iniciais.

    **Etapas:**
    1. Validação dos participantes fornecidos
    2. Criação do grupo no WhatsApp
    3. Adição automática dos participantes
    4. Configuração de permissões iniciais
    5. Retorno das informações do grupo

    **Importante:** Você será automaticamente definido como administrador do grupo.
  </Accordion>

  <Accordion title="Requisitos de Participantes" icon="user-check">
    Regras para adicionar participantes ao grupo.

    **Requisitos:**
    - Mínimo de 1 participante além do criador
    - Números devem estar no formato internacional
    - Apenas dígitos, sem símbolos ou formatação
    - Participantes devem ter WhatsApp ativo
    - Não podem estar bloqueando você

    **Formato:** `5511999999999` (código país + DDD + número)
  </Accordion>

  <Accordion title="Resposta Detalhada" icon="circle-info">
    A resposta inclui informações completas sobre o grupo criado.

    **Dados retornados:**
    - ID do grupo (JID)
    - Nome do grupo
    - Lista de participantes adicionados com sucesso
    - Lista de participantes que falharam (se houver)
    - Timestamp de criação
    - Status de cada operação

    Permite identificar problemas na adição de participantes.
  </Accordion>

  <Accordion title="Limitações e Comportamento" icon="triangle-exclamation">
    Restrições aplicadas durante a criação.

    **Limitações:**
    - Mínimo: 1 participante além do criador
    - Máximo: 1024 participantes (limite do WhatsApp)
    - Nome do grupo: máximo 25 caracteres
    - Participantes inválidos não impedem a criação

    **Comportamento:** Grupo é criado mesmo se alguns participantes falharem.
  </Accordion>
</AccordionGroup>

### Authorizations

<ParamField path="token" type="string" required>
  Token de autenticação de conexão com a API. <br />
  Exemplo: `zt_BppKZWtzvj1X2SbJAuOSkA0Z63uDBQ0wkb28NJaloKeOUnHk`
</ParamField>

### Body Parameters

<ParamField path="name" type="string" required>
  Nome do grupo. <br />
  Máximo de 25 caracteres. <br />
  Exemplo: `"Equipe de Vendas 2024"`
</ParamField>

<ParamField path="participants" type="array" required>
  Lista de números de telefone dos participantes iniciais. <br />
  Formato internacional, apenas dígitos. <br />
  Mínimo de 1 participante. <br />
  Exemplo: `["5511999999999", "5511888888888"]`
</ParamField>

<Tip>
  **Dica:** Mesmo que alguns participantes falhem ao serem adicionados, o grupo será criado. Verifique a resposta para identificar quais participantes foram adicionados com sucesso.
</Tip>

<RequestExample>

```bash cURL
curl -X POST https://zapclass.com.br/api/group/create \
  -H "Content-Type: application/json" \
  -H "Token: zt_BppKZWtzvj1X2SbJAuOSkA0Z63uDBQ0wkb28NJaloKeOUnHk" \
  -d '{
    "name": "Equipe de Vendas",
    "participants": ["5511999999999", "5511888888888"]
  }'
```

```javascript JavaScript
const response = await fetch("https://zapclass.com.br/api/group/create", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    Token: "zt_BppKZWtzvj1X2SbJAuOSkA0Z63uDBQ0wkb28NJaloKeOUnHk",
  },
  body: JSON.stringify({
    name: "Equipe de Vendas",
    participants: ["5511999999999", "5511888888888"]
  }),
});

const data = await response.json();
if (data.success) {
  console.log("Grupo criado:", data.group.name);
  console.log("ID do grupo:", data.group.id);
  console.log("Participantes adicionados:", data.participants.success.length);
}
```

```php PHP
<?php
$ch = curl_init('https://zapclass.com.br/api/group/create');

$payload = [
    "name" => "Equipe de Vendas",
    "participants" => ["5511999999999", "5511888888888"]
];

curl_setopt($ch, CURLOPT_POST, 1);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($payload));
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Content-Type: application/json',
    'Token: zt_BppKZWtzvj1X2SbJAuOSkA0Z63uDBQ0wkb28NJaloKeOUnHk'
]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$data = json_decode($response, true);

if ($data['success']) {
    echo "Grupo criado: {$data['group']['name']}\n";
    echo "ID: {$data['group']['id']}\n";
    echo "Participantes adicionados: " . count($data['participants']['success']);
} else {
    echo "Erro: " . $data['message'];
}
curl_close($ch);
?>
```
