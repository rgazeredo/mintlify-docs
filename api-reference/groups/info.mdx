---
title: "Informações do Grupo"
api: "POST /group/info"
description: "Obter informações detalhadas de um grupo do WhatsApp"
---

Recupera informações completas de um grupo, incluindo detalhes, participantes, configurações e opcionalmente o link de convite e solicitações pendentes.

## Informações Disponíveis

<AccordionGroup>
  <Accordion title="Detalhes do Grupo" icon="circle-info">
    Informações básicas e identificação do grupo.

    **Dados incluídos:**
    - ID do grupo (JID)
    - Nome do grupo
    - Descrição
    - Foto de perfil (URL)
    - Criador do grupo
    - Data de criação
    - Timestamp da última atualização

    Informações essenciais para identificação e gestão.

  </Accordion>

  <Accordion title="Lista de Participantes" icon="users">
    Informações completas sobre todos os membros do grupo.

    **Para cada participante:**
    - JID (identificador único)
    - Número de telefone
    - Nome
    - Se é administrador
    - Se é super administrador
    - Data de entrada no grupo

    Permite gerenciar permissões e visualizar hierarquia.

  </Accordion>

  <Accordion title="Configurações do Grupo" icon="gear">
    Preferências e restrições configuradas no grupo.

    **Configurações disponíveis:**
    - `announce`: Apenas admins podem enviar mensagens
    - `restrict`: Apenas admins podem editar info do grupo
    - `memberAddMode`: Modo de aprovação para novos membros
    - `ephemeralDuration`: Duração das mensagens temporárias
    - `isCommunity`: Se faz parte de uma comunidade

    Útil para entender permissões e comportamento do grupo.

  </Accordion>

  <Accordion title="Recursos Opcionais" icon="sliders">
    Informações adicionais mediante solicitação.

    **Link de convite** (`getInviteLink: true`):
    - URL completa do link de convite
    - Código do convite
    - Permite compartilhar acesso ao grupo

    **Solicitações pendentes** (`getRequestsParticipants: true`):
    - Lista de números aguardando aprovação
    - Útil para grupos com aprovação de entrada

    **Atualização forçada** (`force: true`):
    - Ignora cache e busca dados atualizados
    - Use quando precisar de informações em tempo real

  </Accordion>
</AccordionGroup>

### Authorization

<ParamField path="api_key" type="string" required>
  Chave de autenticação da sua conexão, utilizada no header Authorization como
  **Bearer** Token. <br />
  Você encontra sua API Key no botão **API** da sua conexão no painel ZapClass. <br />
  A chave sempre começa com o prefixo `zc_`. <br />
  Exemplo: `zc_6FrX2qjM4bHT9pR6cC0sD8wLkG3nE2tZfV5uN1aP`
</ParamField>

### Body Parameters

<ParamField path="groupjid" type="string" required>
  Identificador único do grupo (JID). <br />
  Formato: `número@g.us` <br />
  Exemplo: `120363153742561022@g.us`
</ParamField>

<ParamField path="getInviteLink" type="boolean">
  Recuperar link de convite do grupo. <br />
  Requer permissão de administrador. <br />
  Padrão: `false` <br />
  Exemplo: `true`
</ParamField>

<ParamField path="getRequestsParticipants" type="boolean">
  Recuperar lista de solicitações pendentes. <br />
  Útil para grupos com aprovação de entrada. <br />
  Padrão: `false` <br />
  Exemplo: `true`
</ParamField>

<ParamField path="force" type="boolean">
  Forçar atualização ignorando cache. <br />
  Use para obter dados mais recentes. <br />
  Padrão: `false` <br />
  Exemplo: `true`
</ParamField>

<Info>
  **Nota:** Para obter o link de convite, você precisa ser administrador do
  grupo.
</Info>

```javascript JavaScript - Com análise detalhada
const response = await fetch("https://zapclass.com.br/api/group/info", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    Authorization: "Bearer zc_6FrX2qjM4bHT9pR6cC0sD8wLkG3nE2tZfV5uN1aP",
  },
  body: JSON.stringify({
    groupjid: "120363153742561022@g.us",
    getInviteLink: true,
    getRequestsParticipants: true,
  }),
});

const data = await response.json();

if (data.success) {
  const group = data.group;

  console.log("=== INFORMAÇÕES DO GRUPO ===");
  console.log(`Nome: ${group.name}`);
  console.log(`Descrição: ${group.description || "Sem descrição"}`);
  console.log(`Criador: ${group.creator}`);

  console.log("\n=== PARTICIPANTES ===");
  const admins = group.participants.filter((p) => p.admin);
  const members = group.participants.filter((p) => !p.admin);
  console.log(`Total: ${group.participants.length}`);
  console.log(`Administradores: ${admins.length}`);
  console.log(`Membros: ${members.length}`);

  console.log("\n=== CONFIGURAÇÕES ===");
  console.log(`Apenas admins enviam: ${group.announce ? "Sim" : "Não"}`);
  console.log(`Apenas admins editam: ${group.restrict ? "Sim" : "Não"}`);

  if (group.inviteLink) {
    console.log("\n=== LINK DE CONVITE ===");
    console.log(group.inviteLink);
  }

  if (group.pendingRequests && group.pendingRequests.length > 0) {
    console.log("\n=== SOLICITAÇÕES PENDENTES ===");
    console.log(`Total: ${group.pendingRequests.length}`);
  }
}
```

```javascript JavaScript - Listar administradores
async function getGroupAdmins(groupJid) {
  const response = await fetch("https://zapclass.com.br/api/group/info", {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      Authorization: "Bearer zc_6FrX2qjM4bHT9pR6cC0sD8wLkG3nE2tZfV5uN1aP",
    },
    body: JSON.stringify({ groupjid: groupJid }),
  });

  const data = await response.json();

  if (!data.success) {
    throw new Error(data.message);
  }

  const admins = data.group.participants.filter((p) => p.admin);

  return admins.map((admin) => ({
    jid: admin.jid,
    number: admin.number,
    isSuperAdmin: admin.superAdmin || false,
  }));
}

// Uso
const admins = await getGroupAdmins("120363153742561022@g.us");
console.log("Administradores:", admins);
```

```php PHP - Buscar informações
<?php
$ch = curl_init('https://zapclass.com.br/api/group/info');

$payload = [
    "groupjid" => "120363153742561022@g.us",
    "getInviteLink" => true
];

curl_setopt($ch, CURLOPT_POST, 1);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($payload));
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Content-Type: application/json',
    'Authorization: Bearer zc_6FrX2qjM4bHT9pR6cC0sD8wLkG3nE2tZfV5uN1aP'
]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$data = json_decode($response, true);

if ($data['success']) {
    $group = $data['group'];

    echo "Nome: {$group['name']}\n";
    echo "Participantes: " . count($group['participants']) . "\n";
    echo "Criado em: " . date('d/m/Y', $group['createdAt'] / 1000) . "\n";

    if (isset($group['inviteLink'])) {
        echo "Link: {$group['inviteLink']}\n";
    }
} else {
    echo "Erro: " . $data['message'];
}

curl_close($ch);
?>
```
